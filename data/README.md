# Data Directory

This folder contains two datasets repository used for this research.
The Tuberculosis (TB) Chest X-ray Database "https://www.kaggle.com/datasets/tawsifurrahman/tuberculosis-tb-chest-xray-dataset" used for training, validation, and internal testing.
The Shenzhen dataset: https://www.kaggle.com/datasets/raddar/tuberculosis-chest-xrays-shenzhen used for external testing.
The Montgomery dataset https://www.kaggle.com/datasets/raddar/tuberculosis-chest-xrays-montgomery used for external testing 
Subfolders:
- Tuberculosis/
- Normal/
